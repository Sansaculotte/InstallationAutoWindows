Installation Automatisée de Windows

Description :

Un outil simple, mais assez robuste,

pour installer plusieurs programmes et exécuter des scripts powershell ou d'invite de commande

pour les machines Windows 10 et plus récentes.

Il peut être entièrement automatisé à un degré assez élevé

et d'autres fonctionnalités sont prévues dans un avenir proche.

J'ai choisi cet outil parce que pour mon DP, j'ai créé un fichier image de Windows10

pour une installation entièrement automatisée -dite Zero Touch- de windows pour nos VM sur Hyper-V
